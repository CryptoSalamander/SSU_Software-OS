
         OS Assignment #1

WHAT IS THIS?
   20150291 소프트웨어학부 가반 하현수의 운영체제 과제 1번 제출본입니다.

ON WHAT HARDWARE DOES IT RUN?
   리눅스 16.04버전에서 작성하였으며, 
   리눅스 16.04버전에서 구동되는 것을 확인하였습니다.

DOCUMENTATION:
   OS Assignment pdf이 보고서입니다, 명세서에 적혀있는소프트카피가 
   무엇을 의미하는지 감이 잘 오지 않아서, pdf로 변환하기전 원본 docx파일과
   소스코드를 캡처해서 문서화한 파일을 추가로 압축하였습니다.

INSTALLING the source:
   소스코드를 넣으시고 makefile을 통해서 make하시면 정상적으로 작동합니다.
   config1.txt 파일은 제 코드에 맞게끔 수정되어 있으니 
   전혀 다른 형식으로 수정될 시 문제가 발생할 수 있습니다.  

SOFTWARE REQUIREMENTS:
   주석은 Window환경에서 Visual studio 2015를 사용하여 작성하였습니다. 
   Linux에서 한글이 깨지는 문제가 발생할 수 있습니다.

CONFIGURING:
   make를 사용하여 한번에 컴파일이 가능합니다.

COMPILING:
   make를 한번 입력하시면 모든 소스의 오브젝트파일과 실행파일이 생성됩니다.
   그 후 make test를 하시면 정상 작동합니다.
